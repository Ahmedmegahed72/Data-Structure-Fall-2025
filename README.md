# Data Structure - Fall 2025

This repository contains course materials, assignments, and C++ code for the **Data Structure** course in Fall 2025 at BFCI.

## Contents

- Arrays and Pointers
- Linked Lists
- Stacks and Queues
- Trees (Binary Tree, BST)
- Graphs
- Sorting & Searching algorithms

### Notes

- Only source code files and important project files are included.
- Temporary files and build folders (like `.vs/`, `Debug/`, `Release/`) are ignored using `.gitignore`.
